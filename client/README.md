# Chat Client (React)

A Hebrew, RTL chat UI for the task manager server. See the main README in the project root for full setup instructions.

```powershell
npm install
npm run dev
```

The API URL is set via `VITE_API_URL` in the `.env` file (default: `http://127.0.0.1:8000/chat`).
